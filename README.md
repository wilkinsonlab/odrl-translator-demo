# Prerequisites

- Node.js v16.0.0 or higher
- PNPM or NPM or Yarn

# How to use

1. Go to server and run

```sh
pnpm install

# after installation, run:

pnpm run dev
```

2. Go to frontend and run

```sh
pnpm install

# after installation, run:

pnpm run dev
```

3. Now, you can visit http://127.0.0.1:5173/ and test the translator.
